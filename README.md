# Finance Operations & Month-End Reporting Pack

## Project Overview
This project is an Excel-based finance operations and month-end reporting pack created as an independent portfolio project.

The project simulates core finance processes including accounts payable, accounts receivable, cashbook activity, aged balances, overdue invoice tracking, month-end journal entries, and control checks.

The objective was to build a structured reporting pack that improves finance data organisation, traceability, and month-end reporting accuracy.

## Important Note
This is an independent portfolio project. Public company information was used only as a reference point, while invoice-level accounts payable, accounts receivable, cashbook, aging, journal, accrual, and prepayment data were modelled for analysis purposes. The project does not represent internal company data.

The project uses a structured finance dataset including:

- Accounts receivable records
- Cashbook transactions
- Month-end adjustment entries

## Key Features
- Accounts payable and receivable tracking
- Aged payables and aged receivables analysis
- Overdue invoice identification
- Cashbook reconciliation support
- Month-end journal entries for accruals and prepayments
- Control checks to improve data accuracy and consistency
- Finance operations dashboard with KPI summary

## Key Outcomes
- Built an end-to-end finance operations model in Excel
- Improved data organisation and traceability
- Automated aged payable and receivable tracking
- Created a month-end reporting pack suitable for finance review
- Designed control checks to validate reporting accuracy
- Produced dashboard insights for AP, AR, invoice status, and cash movement
